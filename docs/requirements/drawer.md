# drawer

## 功能需求

- [x] drawer 内容 + drawer 遮罩(overlay)
- [x] 内置drawer出现时的过渡动画
  - [x] slide in
- [x] 自定义遮罩的背景颜色
- [x] 自定义drawer出现的方向
  - [x] 水平方向：左或者右
- [x] 关闭drawer的方式
  - [x] 点击drawer内容中标题区域的关闭按钮X
  - [x] 点击遮罩区域
  - [x] 键盘ESC按键
- [ ] 自定义width
- [ ] 是否显示header
  - 当header不显示时，关闭按钮也必须隐藏。此时header不占页面空间。
